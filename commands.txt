Lista Comandi:

	Installare Anchor sulle Ancore cambiando per ognuna i parametri in ApplicationDefinitions

	Installare Blind sul mote da localizzare

	Per compilare ed installare le app sul mote (nella directory dell'app):
		make telosb install bsl,/dev/ttyUSB0

	Per compilare i file .java:
		javac *.java

	Eseguire BlindMain (nella directory java):
		java BlindMain -comm serial@/dev/ttyUSB0:telos
